# DMarketing Homepage

## Overview
DMarketing is a digital marketing services website that offers various services including SEO, email marketing, social media marketing, mobile marketing, affiliate marketing, and pay-per-click (PPC) advertising. This project provides a user-friendly interface for potential clients to learn about services, browse a product catalog, and initiate a checkout process.

## Project Structure
```
dmarketing-homepage
├── index.html          # Main structure of the webpage
├── styles              # Contains CSS files
│   └── main.css        # Styles for the webpage
├── scripts             # Contains JavaScript files
│   ├── main.js         # Main JavaScript for interactions
│   └── cart.js         # Add-to-cart functionality with session-based tracking
└── README.md           # Project documentation
```

## Features
- **Responsive Design**: The website is designed to be mobile-friendly and responsive across various devices.
- **Service Overview**: Detailed sections for each digital marketing service offered.
- **Product Catalog**: Users can search and filter marketing products by category.
- **Add-to-Cart Functionality**: Users can add products to their cart, update quantities, and proceed to checkout.
- **Checkout Modal**: A modal for processing payments through PayPal and Stripe.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the website.
3. Ensure that you have an internet connection to load external resources (e.g., PayPal SDK).

## Usage Guidelines
- Navigate through the website using the header links.
- Explore the services and product catalog.
- Use the search and filter options in the product catalog to find specific items.
- Add items to your cart and proceed to checkout using the modal.
