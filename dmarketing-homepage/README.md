# DMarketing Homepage

Welcome to the DMarketing homepage project! This document provides an overview of the project structure, setup instructions, and features.

## Project Structure

```
dmarketing-homepage
├── index.html         # Main HTML document for the DMarketing homepage
├── styles             # Directory for CSS styles
│   └── main.css       # CSS styles for the homepage
├── scripts            # Directory for JavaScript files
│   └── main.js        # JavaScript for interactive features
├── assets             # Directory for assets
│   ├── fonts          # Custom fonts used in the project
│   └── svgs           # SVG files for icons and graphics
└── README.md          # Project documentation
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd dmarketing-homepage
   ```

3. **Open `index.html` in your web browser** to view the homepage.

## Features

- Responsive design that adapts to various screen sizes.
- Custom typography using fonts from the `assets/fonts` directory.
- Interactive elements powered by JavaScript in `scripts/main.js`.
- SVG graphics for a modern look and feel.
